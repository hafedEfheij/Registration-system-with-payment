🔧 الحلول المطبقة

1. إزالة ::before وإضافة white-space:

.card-status-not-sold { /* إزالة position: relative */ white-space: nowrap; /* منع كسر السطر */ gap: 4px; /* تقليل المسافة */ } /* حذف ::before تماماً */

2. دمج الأيقونة مع النص:

// OLD const saleStatusText = isSold ? 'مباع' : 'غير مباع'; // NEW const saleStatusText = isSold ? 'مباع' : '❌ غير مباع';

3. تحسين عرض الشارة:

<span class="${saleStatusClass}"> ${isSold ? '<i class="fas fa-check-circle"></i>' : ''} ${saleStatusText} </span>
