🔧 التغييرات المطبقة

1. توحيد حجم الأزرار:

// إضافة btn-sm لجميع الأزرار const receiptButton = `<button class="btn btn-primary btn-sm me-2 mb-2">`; const unenrollButton = `<button class="btn btn-danger btn-sm">`; const disabledButton = `<button class="btn btn-secondary btn-sm" disabled>`;

2. تقصير النصوص الطويلة:

// OLD - نص طويل "إلغاء التسجيل (مدفوع)" // NEW - نص مختصر ✅ "مدفوع"

3. الكود النهائي:

const receiptButton = course.payment_status !== 'خالص' ? `<button class="btn btn-primary btn-sm me-2 mb-2 submit-receipt-button"> <i class="fas fa-money-check-alt me-1"></i> إدخال رقم الإيصال </button>` : ''; const isPaid = course.payment_status === 'خالص' || course.payment_status === 'paid'; const unenrollButton = isPaid ? `<button class="btn btn-secondary btn-sm" disabled title="لا يمكن إلغاء التسجيل للمواد المدفوعة"> <i class="fas fa-lock me-1"></i>مدفوع </button>` : `<button class="btn btn-danger btn-sm unenroll-button" data-id="${course.id}"> <i class="fas fa-times me-1"></i>إلغاء التسجيل </button>`;
